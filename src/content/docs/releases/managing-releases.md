---
title: Managing Releases
description: How to create, edit, and organize your music releases in ReleaseLoop.
---

Releases are the core of ReleaseLoop. Each release represents a single, EP, or album and contains tracks, tasks, marketing activities, financials, and assets.

## Creating a release

1. Click **New Release** from the releases page or dashboard
2. Fill in the details:
   - **Title** -- the release name
   - **Primary artist(s)** -- select from your artist roster
   - **Release date** -- the planned or actual release date
   - **Type** -- Single, EP, or Album
   - **UPC** -- Universal Product Code (optional)
   - **Catalogue number** -- your internal reference (optional)
3. Click **Create**

## Viewing releases

The releases page offers two view modes:

- **Grid view** -- card layout showing cover art, title, status, artists, and release date
- **Board view** -- Kanban-style columns organized by release status

Use the search bar and filters to find specific releases.

## Release statuses

Releases move through workflow statuses:

| Status | Meaning |
|--------|---------|
| **Draft** | Initial setup, not yet finalized |
| **Scheduled** | Confirmed and planned for a future date |
| **Released** | Live and available on platforms |
| **Archived** | Completed or no longer active |

## Editing a release

Click on any release to open its detail page. From there you can:

- Edit metadata (title, artists, dates, UPC, catalogue number)
- Update the release status
- Set or change the cover art
- Access all tabs: Tracks, Tasks, Financials, Marketing, Assets, Comments, Activity

## Deleting a release

Releases can be deleted by workspace admins from the release detail page. This action is permanent.

## Release detail tabs

Each release has dedicated tabs for managing different aspects:

- [Tracks](/releases/tracks/) -- songs on the release
- [Tasks](/releases/tasks/) -- checklist of things to do
- [Financials](/financials/budgets/) -- budget and revenue tracking
- [Marketing](/releases/marketing/) -- promotional activities
- [Assets](/releases/assets/) -- files and artwork
- [Comments](/releases/comments/) -- team discussion and activity log
