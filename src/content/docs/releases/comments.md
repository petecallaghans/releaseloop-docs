---
title: Comments & Activity
description: How to collaborate on releases using comments and view the activity feed.
---

Each release has **Comments** and **Activity** tabs for team collaboration and tracking changes.

## Comments

The Comments tab is a discussion thread for your team to communicate about a release.

- Post messages visible to all workspace members with access
- Use **@mentions** to tag specific team members
- Comments are displayed in chronological order

This is useful for sharing feedback on artwork, coordinating launch timing, or leaving notes for collaborators.

## Activity feed

The Activity tab shows a chronological log of everything that's happened on a release:

- Task completions
- Status changes
- File uploads
- Marketing activity updates
- Team member actions

The activity feed is read-only and provides a full audit trail of the release's history.
