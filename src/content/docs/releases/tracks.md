title: Tracks
description: How to add and manage tracks on a release.
---

The **Tracks** tab on a release lets you manage the individual songs that make up the release.

## Adding tracks

1. Open a release and go to the **Tracks** tab
2. Click **Add Track**
3. Fill in the track details:
   - **Title** -- the song name
   - **ISRC** -- International Standard Recording Code (used for royalty matching)
   - **ISWC** -- International Standard Musical Work Code (optional)
   - **Track number** -- position on the release
   - **Artist credits** -- performers on this specific track

## Why ISRCs matter

ISRCs are critical for the royalty import feature. When you import distributor CSV files, ReleaseLoop matches royalty rows to tracks using their ISRC codes. Adding ISRCs to all your tracks ensures accurate financial reporting.

## Editing and removing tracks

- Click on any track to edit its details
- Reorder tracks by changing track numbers
- Remove tracks that are no longer part of the release

## Track metadata

Each track can store:

- Title and track number
- ISRC and ISWC codes
- Contributing artists (may differ from the release artist)
